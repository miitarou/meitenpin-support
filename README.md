# MeitenPin Support

This public repository is the support tracker for MeitenPin / 名店ピン.

Use Issues for:

- Bug reports
- Feedback
- Restaurant data correction requests
- App Store review support checks

Issues are public. Do not include personal information, reservation details,
phone numbers, precise current location, or other private information.

MeitenPin is a free, unofficial fan tool and is not affiliated with Tabelog,
Kakaku.com, or the Tabelog Hyakumeiten program.
