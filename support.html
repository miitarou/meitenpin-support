日本語

名店ピンに関する不具合報告、ご意見、店舗情報の修正希望は、公開サポート用GitHub Issuesからご連絡ください。

お問い合わせ前に

  • GitHub Issuesへの投稿にはGitHubアカウントが必要です。
  • 投稿内容は公開されます。個人情報、予約情報、正確な現在地などは書き込まないでください。
  • 店舗の営業時間、定休日、予約可否、最新の掲載状況は、来店前に公式サイトや各店舗の情報をご確認ください。

English

For bug reports, feedback, or restaurant data correction requests about MeitenPin, please contact us through the public GitHub Issues support tracker.

Before Contacting Support

  • A GitHub account is required to post an issue.
  • Issues are public. Please do not include personal information, reservation details, exact current location, or other private information.
  • Please check official restaurant sources before visiting for opening hours, holidays, reservations, and the latest listing status.

简体中文

如需报告 MeitenPin 的问题、提供反馈或申请修正餐厅信息，请通过公开的 GitHub Issues 支持页面联系我们。

联系支持前请注意

  • 发布 Issue 需要 GitHub 账号。
  • Issue 内容是公开的。请不要填写个人信息、预约信息、精确当前位置或其他隐私信息。
  • 到店前，请通过官方网站或各餐厅发布的信息确认营业时间、定休日、预约可否和最新入选情况。
